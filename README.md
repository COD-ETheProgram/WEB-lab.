# Fishing Shop — Адаптивний WEB-застосунок

Адаптивний веб-сайт магазину товарів для рибалки, розроблений у межах практичної роботи з дисципліни **«Інтернет-технології та проєктування WEB-застосувань»**.

Проєкт демонструє використання семантичної HTML-верстки, адаптивного дизайну, Flexbox-сітки, медіа-запитів та базової інтерактивності за допомогою JavaScript.

## Функціональні можливості

- Семантична HTML5-верстка
- Адаптивний інтерфейс для різних пристроїв
- Каталог товарів магазину рибалки
- Навігаційне меню з burger-версією для мобільних пристроїв
- Hover-ефекти та CSS-анімації
- Чітка та логічна файлова структура

## Використані технології

- **HTML5** — структура веб-сторінки
- **CSS3** — стилізація, Flexbox, media queries, анімації
- **JavaScript** — реалізація burger-меню
- **Flexbox** — адаптивна сітка
- **Media Queries** — підтримка різних розмірів екранів

## Структура проєкту

fishing-shop/
index.html
css/style.css
js/menu.js
assets/img/

## Адаптивність

Інтерфейс оптимізований для таких розмірів екранів:
- **1200px** — десктопна версія
- **768px** — планшети
- **480px** — смартфони

На мобільних пристроях навігація реалізована у вигляді burger-меню.

## Як запустити проєкт

1. Скачати файли з репозиторію.
2. Відкрити файл index.html у будь-якому сучасному браузері.
